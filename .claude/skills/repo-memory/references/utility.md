# Utility commands

## `/repo-memory`

Read `REPO_MEM_DIR/MEMORY.md` and count the memories. Display the available subcommands with the actual count, e.g.:

```
list (3 memories)
audit
```

## `/repo-memory list`

List each memory in `REPO_MEM_DIR` with its name, age, path, type, and contents (full body if short, a one-sentence summary otherwise). Group by type.
