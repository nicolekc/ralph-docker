# Utility commands

Read memories from `REPO_MEM_DIR`.

## `/repo-memory`

Display the available subcommands with memory counts:

```
list (3)
audit
show <name>
```

## `/repo-memory list`

List each memory grouped by type (from frontmatter), with descriptions.

## `/repo-memory show <name>`

Print a memory's full content. Resolve partial names if unambiguous.
